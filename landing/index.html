User Manual

Getting Started

  1. Open Solo in your web browser or download the desktop app
  2. Your data is automatically migrated to the new database system for better performance
  3. Create your first notebook using the menu in the sidebar
  4. Start writing by clicking the "+" button in the toolbar

Search and Discovery

  • Access search from the sidebar menu or use the search button
  • Use fuzzy search to find notes even with typos or partial matches
  • Add tag filters with AND/OR/NOT operators for precise filtering
  • Search within paragraph tags for content-specific results
  • View matching paragraphs with context indicators
  • Click any note in search results to open it directly

Writing and Formatting

  • Use the toolbar buttons to:
    • Add images
    • Insert links
    • Create task lists
    • Mark text as censored
    • Add tags to paragraphs
  • Format text using headings (H1-H3)
  • Create task lists with checkboxes
  • Auto Zen mode activates when you write more than 5 words
  • Tag paragraphs for better organization and searchability
  • Use speech-to-text for hands-free writing
  • Insert cut-ins for side notes and annotations

Privacy Features

  • Set up censorship PIN in Settings → Censorship
  • Mark text as censored using the toolbar button or Ctrl+Alt+X
  • Mark entire notes or notebooks as censored
  • Quick-enable censorship with Ctrl+.
  • When censorship is enabled, press Ctrl+. to quickly access settings
  • Disable censorship by entering your PIN
  • Note and notebook-level censorship for granular control

Organization

  • Create nested notebooks by selecting a parent notebook
  • Add hierarchical tags using forward slashes (e.g., "work/projects/active")
  • Tag paragraphs for content organization
  • Move notes between notebooks using note settings
  • Pin/unpin sidebar for quick access
  • Use advanced search to find notes across all notebooks

Timeline View

  • Access timeline view from the sidebar menu
  • Tag notes or paragraphs with "Main events" or "Главные события" to include them in timeline
  • Navigate through years and months with smooth scrolling
  • Use the date picker in the header to jump to specific months
  • View notes in chronological order with callout-style connections
  • Click any timeline note to open it directly
  • Timeline automatically scrolls to current month on open

Synchronization

  • Choose your sync method:
    • WebDAV for self-hosted storage
    • Server for centralized storage
  • Quick-sync changes with Ctrl+S
  • Automatic backup of your data
  • Restore from previous backups
  • Improved sync performance with optimized data structure

Keyboard Shortcuts

Ctrl + Alt + X Toggle censored text
Ctrl + . Enable censorship / Open settings
Ctrl + S Sync changes
Ctrl + K Open search
Ctrl + T Open timeline
Ctrl + Z Undo
Ctrl + Shift + Z Redo
